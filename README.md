# Caesar Cipher Project 🔐

This is my first ever "mini project" as a cyber security student learning python.
It is a simple Python script that encrypts and decrypts text using a classic Caesar Cipher. I am actively learning how python as a programming language is used for automation and system management and task handling in the world of Cyber Security to improve my skills. Although this is a very basic project, overtime ill be adding more improvements the more I learn. 

## Features
* **Encryption:** Converts plaintext to ciphertext using a fixed shift.
* **Decryption:** Reverses the process to reveal the original message.
* **Error Handling:** Identifies and flags non-alphabetic characters.

## How to Run
1. Ensure you have Python installed.
2. Run the script: `python Caesar_cipher.py`
3. Follow the prompts in terminal to encode or decode.
